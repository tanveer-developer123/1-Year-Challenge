CSS Variables (:root) Demo

This page demonstrates how to use CSS Variables with :root for easy theme control and reusability.

:root { --primary-color: #3498db; --secondary-color: #2ecc71; --text-color: #333; --bg-color: #f4f4f4; /* Dark Theme */ --dark-primary: #1abc9c; --dark-secondary: #e67e22; --dark-text: #f1f1f1; --dark-bg: #222; }

Why use CSS Variables?

  • ✅ Reusability: Define a value once, use it everywhere with var().
  • ✅ Easy updates: Change one variable in :root, and it updates across the site.
  • ✅ Themes: Quickly switch between light/dark mode by changing variable values.
  • ✅ Maintainability: Cleaner and more organized code.
