CSS Core Concepts

Master the fundamentals of CSS styling

1. CSS Syntax (Selectors, Properties, Values)

h1
{
color: #2c3e50;
font-size: 2rem;
text-align: center;
}

Syntax Components:

  • Selector: Targets HTML elements (h1, .class, #id)
  • Property: What to style (color, font-size, etc.)
  • Value: How to style it (#2c3e50, 2rem, center)

2. Box Model (Content, Padding, Border, Margin)

Content

Box Model Properties:

.content {
  width: 200px;
  height: 100px;
  padding: 20px;
  border: 3px solid #e74c3c;
  margin: 30px;
}

3. CSS Units (px, %, em, rem, vh, vw)

Pixel (px) - Fixed Size

20px

Fixed size, doesn't scale with parent

Percentage (%) - Relative to Parent

50%

50% of parent width

Em (em) - Relative to Font Size

2em

2x current font size

Rem (rem) - Relative to Root Font

1.5rem

1.5x root font size

Viewport Height (vh)

10vh

10% of viewport height

Viewport Width (vw)

15vw

15% of viewport width

5. Specificity & Inheritance

Specificity Hierarchy:

1 Inline styles (style="")
2 ID selectors (#id)
3 Class selectors (.class)
4 Element selectors (p, h1)

Inheritance Demo:

Parent (color: blue)
Child inherits color
Grandchild also inherits

6. Shorthand Properties

Margin Shorthand

Long Form:
margin-top: 10px;
margin-right: 20px;
margin-bottom: 10px;
margin-left: 20px;
Shorthand:
margin: 10px 20px;

Background Shorthand

Long Form:
background-color: #3498db;
background-image: url('image.jpg');
background-repeat: no-repeat;
background-position: center;
Shorthand:
background: #3498db url('image.jpg') no-repeat center;

Border Shorthand

Long Form:
border-width: 2px;
border-style: solid;
border-color: #e74c3c;
Shorthand:
border: 2px solid #e74c3c;
