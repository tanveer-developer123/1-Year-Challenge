Pseudo Elements & Selectors Mastery

Advanced CSS tricks for content, interaction, and targeting

1. ::before & ::after (Content Injection)

Item with ::before & ::after
.label::before {
  content: "★ ";
  color: #fd79a8;
}
.label::after {
  content: " ✔";
  color: #00b894;
}

2. :hover & :focus (Interaction)

.hover-btn:hover,
.hover-btn:focus {
  background: #00b894;
  box-shadow: 0 0 12px #00b89466;
}

3. Advanced Targeting (:nth-child, :not(), :is(), :where())

.advanced-list li:nth-child(odd) {
  background: #fdcb6e;
}
.advanced-list li:not(:first-child) {
  border-left: 3px solid #0984e3;
}
.advanced-list :is(li:nth-child(2), li:nth-child(4)) {
  background: #00b894;
}
.advanced-list :where(li:last-child) {
  font-style: italic;
  background: #fd79a8;
}
