Template and Slot Demo

This section demonstrates how <template> and <slot> work in HTML.

Code Example (Displayed)


<template id="my-template">
  <div class="template-box">
    <h2> This is rendered from a template</h2>
    <p><slot>This is slot content inside the template.</slot></p>
  </div>
</template>

<script>
  const template = document.getElementById("my-template");
  const clone = template.content.cloneNode(true);
  document.getElementById("output").appendChild(clone);
</script>
