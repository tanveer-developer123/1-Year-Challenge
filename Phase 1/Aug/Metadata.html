Metadata Content / <head> / <meta>

This Metadata is used to in html in head section body of out and html tag in innner and is used to a provide information about the document. Is Tag is used to define the character set for the HTML document, which is essential for proper text rendering.

Metadata is not displayed on the page but is machine-readable and can be used by browsers, search engines, and other services.

Metadata can include information such as the document's title, description, keywords, author, and more.

It is important for SEO (Search Engine Optimization) as it helps search engines understand the content of the page.

Metadata can also include links to stylesheets, scripts, and other resources that are required for the proper rendering and functionality of the page.

Example of Metadata in HTML


<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document Title</title>
<meta name="description" content="This is a sample HTML document with metadata.">
<meta name="keywords" content="HTML, metadata, example">
<meta name="author" content="Your Name">
</head>

<body>

</body>

</html>

Topic of <title>

The <title> tag is used to define the title of the HTML document. This title is displayed in the browser's title bar or tab and is also used by search engines as the main title for the page in search results.

It is important for SEO and user experience, as it gives users an idea of what the page is about before they click on it.

The title should be concise, descriptive, and relevant to the content of the page

Topic of <link>

The <link> tag is used to link external resources to the HTML document, such as stylesheets, icons, and other files. It is typically placed within the <head> section of the document.

Here is an example of how to use the <link> tag to include a stylesheet:

<link rel="stylesheet" href="styles.css">

In this example, the <link> tag is used to link to an external CSS file named "styles.css". The "rel" attribute specifies the relationship between the current document and the linked resource, which in this case is "stylesheet".

The <link> tag can also be used to link to other types of resources, such as icons or alternate stylesheets. For example, to link to a favicon (the small icon displayed in the browser tab), you can use the following code: 

<link rel="icon" type="image/x-icon" href="favicon.ico">

Topic of <style>

The <style> tag is used to define internal CSS styles for the HTML document. It is typically placed within the <head> section and allows you to apply styles directly to the elements of the page.

Here is an example of how to use the <style> tag:

<style>
    body {
        background-color: #f0f0f0;
        font-family: Arial, sans-serif;
    }
    h1 {
        color: #333;
    }
    </style>

Topic of <script>

The <script> tag is used to include JavaScript code in the HTML document. It can be placed either in the <head> section or at the end of the <body> section, depending on when you want the script to be executed. 

<script>
    console.log("Hello, World!");
</script>

This is where you can write your JavaScript code. It will be executed when the page loads.
